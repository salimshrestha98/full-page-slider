== Installation ==

1. Upload the plugin ZIP file via the **Plugins > Add New > Upload Plugin** section in your WordPress dashboard.
2. Activate the plugin from the Plugins menu.
3. In the Block Editor, search for **Full Page Slider** in the block inserter.
4. Add the block to your page and begin creating slide sections.

== Frequently Asked Questions ==

= Can I use this with any theme? =
Yes, Full Page Slider works with any modern WordPress theme.

= Is this plugin compatible with page builders like Elementor? =
This plugin is built for the Gutenberg editor. It may not be compatible with other page builders.

= Can I add images, buttons, or videos to slides? =
Yes, each slide supports multiple inner blocks, so you can add almost any content Gutenberg supports.

= Does it support horizontal sliding? =
Yes! You can toggle between vertical and horizontal slide modes.
